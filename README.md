# NyayaSulabh

NyayaSulabh is a web-based platform designed to simplify access to legal information and resources. This project aims to make legal knowledge more accessible and easier to navigate for everyone.

## 🌐 Live Demo

You can view the live project here:

**https://nyaya-sulabh.vercel.app/**

---

## ✨ Features

- 🔐 **User Authentication**
  - Login page (`login.html`)
  - Signup page (`signup.html`)

- ⚖️ **Legal Information Interface**
  - Dedicated interface (`interface.html`) for accessing legal resources.

- 🏠 **Landing Page**
  - Clean and responsive landing page (`index.html`).

- 📱 **Responsive Design**
  - Works across desktop and mobile devices.

- 🌐 **Static Website**
  - Built using HTML and deployed on Vercel.

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Structure of the website |
| CSS3 | Styling |
| JavaScript | Client-side interactivity |
| Vercel | Deployment & Hosting |

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Rishi-314/NyayaSulabh.git
```

### 2. Navigate to the Project

```bash
cd NyayaSulabh
```

### 3. Run the Project

Simply open the `index.html` file in your preferred browser.

No additional setup is required.
